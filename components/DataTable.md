# Data Table

A data table displays structured information in rows and columns, allowing
users to scan, compare, and analyze data. Uses semantic `<table>` elements
with proper `<caption>`, `<thead>`, `<tbody>`, and scope attributes for
accessibility.
# DataTable

- Semantic `<table>` with optional `<caption>`
- `aria-label` used when caption absent
- Children provide thead/tbody structure
