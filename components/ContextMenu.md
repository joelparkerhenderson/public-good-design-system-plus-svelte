# Context Menu

A context menu provides a list of actions relevant to the current context,
typically triggered by a right-click or long-press. The menu appears near the
pointer and offers contextual options such as cut, copy, paste, or custom
actions.
# ContextMenu

- `role="menu"` with `aria-label`
- `$bindable()` open prop controls visibility
- Arrow keys navigate menuitems, wraps around
- Home/End jump to first/last item
- Escape closes the menu
- Focus moves to first menuitem on open
